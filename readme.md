# Natural Language Processing - Knowledge Artifact Repository

## Project Description
This repository serves as a systematic, reusable learning resource for fundamental Natural Language Processing concepts. It fulfills the criteria for the Continuous Comprehensive Evaluation Activity – I for Semester V, A.Y. 2026-27 at Shah & Anchor Kutchhi Engineering College. 

The website captures, organizes, and explains key Week 1 concepts including Text Preprocessing, Feature Engineering, Language Representation, and Language Models.

## Features
- **Zero Frameworks:** Built purely with Semantic HTML5, CSS3 Variables, and Vanilla JavaScript.
- **Glassmorphism UI:** Modern, AI-inspired aesthetic with transparent navbar.
- **Dark Mode:** Integrated native light/dark theme toggle.
- **Interactive Concept Cards:** Includes code snippets with a functional "Copy Code" button, filtering via tabs, and live text search.
- **SVG Workflows:** Lightweight, animated SVG diagrams demonstrating NLP pipelines without relying on heavy external images.
- **Responsive Tables:** Cross-device friendly comparative analysis matrices.
- **Scroll Tracking & Animations:** Native IntersectionObserver API for smooth entrance reveals.
